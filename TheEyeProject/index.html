How it works

The Eye Project started as a sort of experiment in randomness. Originally, it generated a single Eye in the center of the canvas, with random values governing most of the parameters for creation. Later versions became more interactive, and finally I added the automatic tutorial to tell you how to use it.

The project runs best on modern Google Chrome browsers. The Color Painter functionality is an HTML5 device, and I didn't try very hard to make IE or Firefox supported. Sorry. Use Chrome.

So, read the text in the automatic tuorials. It's very helpful I think, if you know the terms:

  1. Eye: The circular shape that is created upon clicking. An Eye has a single pupil, and many rings.
  2. Pupil: The center of the Eye, like the hole in a donut. There are many functions that can only be performed by clicking and holding the pupil you can:
    • Grow the entire Eye itself, linearly scaling the entire thing up or down. Do it with Q and W.
    • Scale the size of the pupil only. Do it with A and S.
    • Increasing or decreasing the number of vertices in the very first ring. If you click on the ring and increase the number of vertices, you actually only grow the second part of the ring's number of vertices- so this function has to go here. Do it with Z and X.
    • Add a new ring to the outside of the Eye. It will be White and will not be tagged for the Color Painter- more on that later. Do it with E.
    • Delete the last ring from the Eye. Do it with D.
    • Reset all of the Color Painter's points. More on that later. Do it with R.
  3. Ring: Each section of triangles that surround each other and the pupil is a ring. The number of rings is random but can be changed manually. See above for that. Each Ring
